# RadioPato-Frontend
Web application to manage the community of neighbors

This project will hold all the packages for the frontend.

## Structure 

```
src/
| components/
|   |-- Register/
|   |   |-- Register.tsx\n
|   |   |-- Register.css
|-- contexts/
|   |-- UserContext/
|   |   |-- UserContext.tsx
|-- hooks/
|   |-- useMediaQuery/
|   |   |-- useMediaQuery.tsx
|-- pages/
|   |-- UserProfile/
|   |   |-- UserProfile.tsx
|   |   |-- UserProfile.css
|   |-- index.js
|-- routes/
|   |-- routes.tsx
|-- utils/
|   |-- some-util/
|   |   |-- someUtil.tsx
|-- services/
|   |-- some-service/
|   |   |-- someService.tsx/
|-- App.tsx
|-- index.tsx
```


## Project Scripts
